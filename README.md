
# K-Means Clustering from Scratch (NumPy Only)

## Project Overview
This project implements the K-Means clustering algorithm **from scratch using NumPy only**.
No high-level clustering APIs are used.

## Contents
- `kmeans_from_scratch.ipynb` – Full Google Colab compatible notebook
- `requirements.txt` – Dependencies

## How to Run
Open the notebook in **Google Colab** or Jupyter Notebook and run all cells sequentially.
